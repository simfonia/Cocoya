# 雙臂角度即時控制範例 (MediaPipe + piCar)

本範例展示了如何使用 PC 端的 USB 攝影機，透過 **MediaPipe Pose (姿態偵測)** 辨識使用者的左肩-左肘與右肩-右肘兩條連線段的夾角，並將該夾角映射為 `0% ~ 100%` 的雙臂張開百分比。該數據經由 **Serial 串列埠** 即時傳送給 **πCar (Maker Pi RP2040)**，即時控制其雙臂的同步張合。

---

## 1. 運作原理說明

### 角度計算與控制映射
* **向量線段與夾角**：PC 端程式取得關鍵點：
  * 左肩 `11`、左肘 `13` $\rightarrow$ 建立左臂向量。
  * 右肩 `12`、右肘 `14` $\rightarrow$ 建立右臂向量。
  * 計算兩向量之夾角 $\theta$。
* **狀態與百分比映射**：
  * **水平以下（張開度 0%）**：當兩側手肘 Y 座標大於肩膀 Y 座標（手垂直朝下放）時，代表雙手在水平線以下，輸出 `0%`（此時車子雙手張開）。
  * **水平以上（張開度 0% ~ 100%）**：當手舉高於水平線以上：
    * 雙手平舉時：兩手臂向量方向相反，夾角 $\theta = 180^\circ$，映射為 `0%` 的張開度。
    * 雙手垂直向上時：兩手臂向量方向相同，夾角 $\theta = 0^\circ$，映射為 `100%` 的張開度（此時車子雙手旋起至最收攏點）。
    * 運算公式為：`percent = (180 - theta) / 180 * 100`。

### 通訊協議
PC 端程式計算出 `percent` 後，直接將數值以文字格式發送：
`百分比數值\n`
例如：`0\n`（雙手放下/平舉）或 `100\n`（雙手垂直高舉）。

## 2. 使用步驟指南

### 步驟 A：準備 πCar 端 (MCU)
1. 開啟 **Cocoya IDE** 並切換至 MicroPython (piCar) 平台。
2. 載入積木專案 [1-1_YPose_piCar_Control.xml](cocoya/examples/人體姿勢控制πCar爪子/1-1_YPose_piCar_Control.xml)。
3. 將 πCar 連接至電腦的 USB 埠。
4. 點擊 **執行/下載** 將程式寫入並在 πCar 上運行。
   * 運行成功時，πCar 的雙手會執行歸位，且蜂鳴器會發出短促提示聲。
   * 關閉 Serial 監控

### 步驟 B：準備 PC 端環境
1. 切換至 **PC 平台**。
2. 載入積木專案 [1-2_YPose_PC_Control.xml](cocoya/examples/人體姿勢控制πCar爪子/1-2_YPose_PC_Control.xml)。
   * 如果您的 πCar 序列埠不是 `COM4`，請修改 **串列埠初始化** 積木中的 Port 欄位，將其修改為您本機的序列埠（例如 Windows 上的 `COM5`，macOS 上的 `/dev/tty.usbmodem...`）。
3. 確保已於終端機安裝必要庫：`pip install opencv-python mediapipe pyserial`
4. 點擊 **執行** 啟動攝影機並開始發送控制數據。

### 步驟 C：即時互動
* 站在鏡頭前，雙手在水平以下時 πCar 雙手臂張開；當雙手抬起並往垂直方向靠攏時，πCar 雙臂會同步旋入，舉得越高，旋入越深！
* 按下鍵盤的 **`Q` 鍵** 即可安全退出 PC 端程式。
