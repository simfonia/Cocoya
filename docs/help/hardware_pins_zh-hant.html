開發板腳位說明 (Pinout Reference)

歡迎使用 Cocoya 硬體控制積木！不同的開發板擁有不同的腳位配置，請參考下方的圖表與說明進行開發。

1. Maker Pi RP2040

這款開發板最大的特色是板子表面的標註非常清晰。每個藍色 LED 與排針旁邊都直接印有對應的腳位名稱（例如 GP0, GP1...）。

提示： 在 Cocoya 中使用時，請選擇 [Pico] 開頭的選項。例如板子上的 0 號燈對應 board.GP0。
  • 數位輸出/入： GP0 ~ GP28。
  • 類比讀入 (ADC)： GP26, GP27, GP28。
  • 內建 NeoPixel： GP18。
  • 內建按鈕： GP20。

3. XIAO ESP32-S3 Sense

XIAO S3 體積精巧，腳位採用 D0, D1... 命名。請參考下方的複用功能對照表：

腳位 (Pin) 主要功能 備註 (Multiplexing)
D0 ~ D3 Digital, ADC, PWM 通用腳位，支援類比讀入。
D4 / D5 Digital, ADC, I2C D4 為 SDA, D5 為 SCL。
D6 / D7 Digital, UART D6 為 TX, D7 為 RX。
D8 ~ D10 Digital, ADC, PWM 通用腳位，支援類比讀入。
重要： ESP32-S3 幾乎所有腳位都支援 PWM，但請優先使用 D0~D3 與 D8~D10 進行一般感測器連接。更詳細的腳位複用資訊請參考 Seeed Studio 官方 Wiki。
