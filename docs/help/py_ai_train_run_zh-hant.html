🦉AI訓練積木 (py_ai_train_run)

功能說明

執行 AI 模型訓練。從資料集學習並產出 TFLite 模型檔案，可用於後續的推論（預測）。

支援四種任務類型：圖像分類、物件偵測、循線偵測、表格資料。

基本參數

欄位說明預設值
資料集 (Dataset)訓練資料所在資料夾路徑。每個子資料夾代表一個分類。dataset/classifier_dataset
模型 (Model)訓練完成後模型檔案的輸出資料夾（自動產生）。model/資料集名稱
Epochs訓練輪數。越多代表學習越久，但過多可能導致過擬合。20
Batch批次大小。每次更新權重時同時處理的圖片張數。32
LR (Learning Rate)學習率。控制每次更新的步長，太大可能無法收斂。0.001
Val Split驗證集比例。從資料集中分割多少比例作為驗證用。0.2 (20%)
Dropout隨機丟棄率。防止過擬合，訓練時隨機丟棄部分神經元。0.2 (20%)
Aug (Augmentation)是否啟用資料擴增。自動對圖片進行翻轉/旋轉/縮放等變換。✔ 啟用
類型 (Type)任務類型：分類器 / 偵測器 / 循線 / 表格分類器
後端 (Backend)訓練位置：自動 / 本地 / 遠端自動

進階參數

欄位說明預設值
Backbone骨幹網路。選擇 MobileNetV2、EfficientNetB0 或 ResNet50。MobileNetV2
Optimizer優化器。選擇 Adam、SGD 或 RMSprop。Adam
DNN Layers自訂隱藏層。逗號分隔神經元數，如 "128,64" 表示兩層。
輸入層（GlobalAveragePooling）與輸出層（Softmax）為自動設定，
使用者只需指定中間隱藏層即可。留空則跳過隱藏層直接到輸出。		128,64
Fine-tune是否解凍骨幹網路進行微調。可提升準確率但需更多資料。☐ 停用
💡 不熟悉這些參數的初學者，保留預設值即可獲得良好訓練結果。

訓練輸出檔案

訓練完成後，在模型輸出資料夾中會產生：

  • *.tflite — 量化模型檔案（用於 MCU 或 Edge TPU）
  • *_f32.tflite — Float32 模型檔案（用於 PC 端 Python 推論）
  • *.keras — Keras 原始模型檔案
  • *_labels.txt — 分類標籤對照表
  • *_training_report.html — 訓練報告（含曲線圖與統計）
  • *_training_history.json — 訓練歷史資料（JSON 格式）
  • *_training_curve.png — Loss/Accuracy 曲線圖

注意事項

✅ 圖片尺寸固定為 224×224，與 MobileNetV2 的 ImageNet 預訓練權重綁定。
✅ 驗證集（Validation Set）同時作為最終模型評估，不另設測試集。原因是教育場景資料量較小，三分割會導致各集合樣本不足，且初學者無需區分驗證/測試的複雜概念。
⚠️ 每個分類至少需要 5 張以上圖片，建議每個分類 20-100 張。
⚠️ 路徑中請避免使用中文字元，以免部分環境報錯。
