📦 取得邊界框積木 (py_ai_get_bbox)

功能說明

從物件偵測的推論結果中提取第一個物件的邊界框座標。

輸入：推論結果（py_ai_model_predict 的輸出，需為 detector 類型）

輸出：Tuple (x1, y1, x2, y2) — 邊界框的左上角與右下角座標。

使用範例

result = predict(frame) # 推論
bbox = get_bbox(result) # 取得邊界框
x1, y1, x2, y2 = bbox # 解包座標
cv2.rectangle(frame, (x1, y1), (x2, y2), (0, 255, 0), 2) # 畫框
💡 此積木僅適用於「偵測器 (detector)」任務類型。
