🤖 初始化模型積木 (py_ai_model_init)

功能說明

載入訓練完成的 TFLite 模型，準備進行推論（預測）。

此積木會自動載入對應的標籤檔，並根據任務類型選擇正確的推論引擎。

參數

欄位說明預設值
模型路徑 (Model Path).tflite 模型檔案或目錄的路徑。model
類型 (Type)任務類型（分類器 / 偵測器 / 循線 / 表格）必須與訓練時一致。分類器

標籤檔對應規則

系統會自動搜尋對應的標籤檔：

  • 如果模型路徑是目錄，搜尋目錄中第一個 .tflite 檔案
  • 標籤檔命名規則：{模型資料夾名稱}_labels.txt
  • 例如：模型在 model/gesture_model/，則標籤檔為 model/gesture_model/gesture_model_labels.txt

使用流程

# 1. 初始化模型（放在程式開頭）
init model "model/gesture_model"

# 2. 在迴圈中推論
result = predict(frame)
label = get_label(result)
confidence = get_confidence(result)

注意事項

✅ 只需初始化一次，請放在主程式開始處。
⚠️ 模型路徑與類型必須與訓練時使用的相符。
⚠️ 確保 TFLite 模型檔案與標籤檔都存在。
