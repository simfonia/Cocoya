Board Pinout Reference

Welcome to Cocoya Hardware Control! Different development boards have different pin configurations. Please refer to the charts below for wiring.

1. Maker Pi RP2040

The best feature of this board is that labels are clearly printed on the board surface. Each blue LED and header has its pin name (e.g., GP0, GP1...) printed right next to it.

Tip: In Cocoya, choose options starting with [Pico]. For example, LED 0 on the board corresponds to board.GP0.
  • Digital I/O: GP0 ~ GP28.
  • Analog In (ADC): GP26, GP27, GP28.
  • Onboard NeoPixel: GP18.
  • Onboard Button: GP20.

3. XIAO ESP32-S3 Sense

XIAO S3 is compact and uses D0, D1... as pin names. Please refer to the multiplexing table below:

Pin Main Functions Notes (Multiplexing)
D0 ~ D3 Digital, ADC, PWM General purpose pins with Analog In.
D4 / D5 Digital, ADC, I2C D4 is SDA, D5 is SCL.
D6 / D7 Digital, UART D6 is TX, D7 is RX.
D8 ~ D10 Digital, ADC, PWM General purpose pins with Analog In.
Important: Most pins on ESP32-S3 support PWM, but please prioritize D0~D3 and D8~D10 for general sensor connections. For more detailed pin multiplexing information, please refer to the Seeed Studio Official Wiki.
