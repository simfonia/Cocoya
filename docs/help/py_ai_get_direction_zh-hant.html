🧭 取得方向積木 (py_ai_get_direction)

功能說明

從循線偵測的推論結果中提取行進方向。

輸入：推論結果（py_ai_model_predict 的輸出，需為 line_follower 類型）

輸出：字串 — 方向 ('left', 'right', 'forward', 'none')

使用範例

result = predict(frame) # 推論
direction = get_direction(result) # 取得方向
if direction == "left":
    car.turn_left()
elif direction == "right":
    car.turn_right()
else:
    car.forward()
💡 此積木僅適用於「循線 (line_follower)」任務類型。
