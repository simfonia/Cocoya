Cocoya Framework API Index

1. 通訊橋樑 (CocoyaBridge)

前端與 Host 端 (VSIX/Tauri) 通訊的核心機制。

1.1 動態能力同步 (Capabilities Sync)

Frontend VSIX

用於在啟動時將 Host 端的環境狀態（如是否連線 SSH）同步至前端。

// Host 端 (extension.ts) 發送
this.panel.webview.postMessage({ 
    command: 'manifestData', 
    capabilities: { isRemoteConnected: true, ... },
    cloudAiEnabled: true,
    ...
});

// 前端接收 (controller.js)
this.handlers.set('manifestData', async (m) => {
    if (this.bridge.updateCapabilities && m.capabilities) {
        this.bridge.updateCapabilities(m.capabilities);
    }
    ...
});

2. Blockly 產生器開發

2.1 使用 definitions_ 解耦定義與執行

為了解決積木擺放順序導致的 Python 語法錯誤（如先呼叫後定義），應將所有宣告型代碼放入 definitions_。

Blockly.Python.forBlock['py_function_def'] = function(block, generator) {
  const name = ...;
  const code = `def ${name}():\n  pass\n`;
  // 注入全域定義區
  generator.definitions_['func_' + name] = code;
  return null; // 該積木在當前位置不產出代碼
};
