技術深挖 (Technical Deep Dive)

1. 智慧硬體同步 (Hardware-Aware Sync)

為解決 ESP32-S3 Native USB 緩衝溢出問題，deploy_mcu.py 改採主動查詢 sys.platform 機制。精準識別硬體後，自動分配 128/1024 bytes 傳輸塊與 0.8/0.1s 同步延遲，兼顧了 S3 的極致穩定性與 RP2040 的高效率。

2. UI 焦點保護機制：Silence Mode

真兇為 Minimap 同步與 Orphan Check 頻繁觸發積木狀態變更，導致 WidgetDiv 單例輸入框關閉。透過判斷 isVisible() 暫停所有背景 UI 任務，達成了流暢的打字體驗，並在點擊「執行」時強制同步視覺狀態。

3. 多檔案燒錄組態化

針對 S3 Sense Factory 韌體，實作了基於 project_config.json 的多段燒錄邏輯（對齊 0x0, 0x8000, 0xe000, 0x10000），並將 esptool 指令全面橫線化（Dash-syntax）以消除棄用警告。
