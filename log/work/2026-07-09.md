# 工作日誌 2026-07-09 (四)

## 今日異動

### [Feature] Dataset Manager 多攝影機選擇功能

**問題**：使用 Dataset Manager 進行 live 採集時，只能使用預設攝影機 (deviceId=0)，無法切換到其他 USB 外接攝影機。

**解決方案**：實作完整的「列舉 → 選擇 → 切換」流程，涵蓋後端 Sidecar、Extension Handler、前端 Sampler 與 UI 元件。

#### 修改檔案

1. **`resources/dataset_manager/camera_service.py`**
   - 新增 `@staticmethod list_cameras(max_test=10)` 靜態方法
   - 依序測試 index 0~9，若 `cv2.VideoCapture(i).isOpened()` 且可讀取 frame 則列入可用清單
   - 回傳 `[{id, name}, ...]` 結構

2. **`resources/dataset_manager/dataset_sidecar.py`**
   - 新增 `listCameras` 指令處理分支
   - 呼叫 `CameraService.list_cameras()` 並回傳攝影機清單

3. **`src/handlers/datasetOps.ts`**
   - 新增 `handleDatasetListCameras()` 方法
   - 透過 Sidecar 發送 `listCameras` 指令，將結果轉發給前端

4. **`src/cocoyaManager.ts`**
   - 新增 `datasetListCameras` 訊息路由

5. **`ui/src/modules/dataset_manager/sampler.js`**
   - `state` 新增 `cameraList: []` 與 `selectedDeviceId: 0`
   - 新增 `listCameras()` 方法，Promise 模式監聽 `datasetCameraListResult`
   - 新增 `setCameraDevice(deviceId)` 方法

6. **`ui/src/modules/dataset_manager/ui_components.js`**
   - `renderSamplerView` 加入「📷 攝影機:」下拉選單與「🔄」重新掃描按鈕
   - 綁定 `onchange` 事件更新 `Sampler.state.selectedDeviceId`
   - 綁定重新整理按鈕呼叫 `Sampler.listCameras()`

7. **`ui/src/modules/dataset_manager/ui_layout.js`**
   - `refreshDynamicPanels()` 中，進入 live 模式時自動呼叫 `Sampler.listCameras()`（僅清單為空時）
   - `onStartCamera` 回調改為使用 `Sampler.state.selectedDeviceId`

## 技術深挖 (Technical Deep Dive)

### 通訊鏈路
```
前端 sampler.js listCameras()
  → Bridge.send('datasetListCameras')
    → cocoyaManager.ts setupMessageListener case 'datasetListCameras'
      → datasetOps.ts handleDatasetListCameras()
        → sidecar.send('listCameras')
          → dataset_sidecar.py listCameras 分支
            → CameraService.list_cameras() 列舉 0~9
          ← {success, cameras: [{id, name}]}
        ← postMessage({command: 'datasetCameraListResult', cameras})
      ← window message handler 解析 cameras
    ← sampler.state.cameraList 更新
  ← UI 下拉選單重新渲染
```

### 列舉策略
- 使用 `cv2.VideoCapture(i)` 依序測試 i=0..9
- 額外呼叫 `cap.read()` 確認真的有畫面（避免空訊號源）
- 解析度資訊附加在名稱中以協助辨識不同攝影機
- 前端的 `cameraList` 若為空，自動在進入 live 模式時呼叫列舉
