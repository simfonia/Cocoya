# 工作日誌 2026-07-11 (六)

## 今日異動

### [Feature] 6 個 MicroPython 入門範例（Cytron Maker Pi RP2040）

**任務**：設計一系列的 Cocoya Python 入門範例，以 MicroPython (MCU 模式) + Cytron Maker Pi RP2040 為主，善用聲光效果（蜂鳴器 + GP0-7 藍色指示燈），教學重點依序為 for 迴圈、while 迴圈、多個同層 if（多選）、if-elif-else（單選）、變數、函式。

#### 新增範例檔案

| 檔案 | 語法主題 | 教學設計 |
|------|---------|---------|
| `examples/basic/03_for_loop_LED.xml` | for 迴圈 | LED 跑馬燈 0→7 亮, 7→0 滅，含 for range 正/反向 |
| `examples/basic/04_while_button_buzzer.xml` | while 迴圈 | while True + if-else 控制蜂鳴器 440Hz 與 LED0 |
| `examples/basic/05_multi_if_buttons.xml` | 多個 if (多選) | 5 個獨立 if 積木，每個各自判斷，可同時亮多盞燈 |
| `examples/basic/06_if_elif_else_priority.xml` | if-elif-else (單選) | 1 個 if-elif-else，條件1 (GP20) 會攔截後面的所有條件 |
| `examples/basic/07_variables_counter.xml` | 變數 | count 初始 -1，按鈕1 +1 亮 LED，按鈕2 -1 滅 LED（累加式進度條） |
| `examples/basic/08_functions_light_show.xml` | 函式 | def blink_all() + def play_beep()，主程式呼叫組合表演 |

#### 設計要點
- 全部使用 `mcu_car_set_led_io`（GP0-7 藍色指示燈），避開刺眼的 NeoPixel RGB
- `py_coding_comment` 正體中文詳細註解
- `py_io_print` 設為 `disabled-reasons="MANUALLY_DISABLED"`，學生可自行啟用除錯
- 範例 05 vs 06 刻意對比「多選（多個 if）」vs「單選（if-elif）」差異
- 僅使用已確認存在於 toolbox 中的積木（Coding 分類 + Loops/Logic/Variables/Functions）
